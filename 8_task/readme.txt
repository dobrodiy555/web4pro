1. Создать страницу с формой. 
В форме должны быть следующие поля: name, surname, email, password, confirm password 

2. Реализовать отправку этой формы методом POST. 
Валидация :
 - все поля обязательны
 - email (xxx@xxx.xxx)
 - email - провести проверку нет ли в массиве сотрудников (PeopleCollection) пользователя с таким емейлом
 - пароль (мин. 8 символов. обязательно должен содержать: большую букву, маленькую букву, цифру, спец-символ)
 - пароли совпадают

В случае ошибки информация должна логироваться в файл в любом формате. 
При неудачной проверке
 - пользователю должен выводиться список ошибок над формой
- после трех неудачных попыток регистрации, скрывать форму на 5 мин и отображать сообщение - “Please try again later”
При успешной проверке :
 - форма должна быть отправлена методом POST
 - на странице вместо изначальной формы должно выводиться сообщение :

"Dear {name} {surname}, your account has been created." - при повторном открытии страницы, вместо формы должно выводиться сообщение :
"Registration already completed."
