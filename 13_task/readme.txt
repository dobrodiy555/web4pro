Необходимо создать Bootstrap таблицу которая будет содержать следующие колонки:
Post Id, Title, Description, Author (Full name), Date.
В таблице должны отображаться данные из БД по 30 записей на страницу.

Необходимо реализовать следующие пункты с помощью AJAX (example) запросов:
Сортировка ASC/DESC по Post Id,
Поиск по слову из Description
Поиск по Title (autocomplete), пользователь вводит первые 3 символа, дальше появляется список возможных вариантов из БД.
Поиск по дате создания поста From / To
Добавить кнопку Load More, при клике на которую будут подгружаться следующие 30 записей в конце таблицы.
