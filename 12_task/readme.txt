Реализовать следующий пример с использованием модуля apache mod_rewrite:
Клиент открывает страницу с картинкой
http://your-domain.local/image-123.png
На картинке вывести из базы значение по ключу 123. Например "Hello Bob"

Клиент открывает страницу с картинкой
http://your-domain.local/image-345.png
На картинке вывести из базы значение по ключу 345. Например "Hello Tom". 
Для надписей на картинках использовать GD библиотеку.

В базе к каждой записи сделать счетчик показов. Счетчик должен отрабатывать только по вызову картинки.
Т.е. есть 2 html файла, в каждом будет img картинка с именем в формате image-N.png, где N уникальный идентификатор.
